#PC Gamer News Scraper
- - -
This app pulls all articles from the news section of PCGamer.com and displays them for you to read and comment on
- - -
- This app is made with the following technologies:
    - MongoDB
    - Mongoose
    - Handlebars
    - Axios
    - AJAX
    - Cheerio
